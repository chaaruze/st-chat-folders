## Features

- Per-character folders - Each character has independent folder sets
- Collapsible folders - Click folder headers to expand or collapse
- Right-click menu - Move chats between folders instantly
- Native theme - Blends with SillyTavern's UI
- Persistent state - Folder states are saved across sessions
- Mobile friendly - Full-screen popup on mobile devices

---

## Installation

1. Open SillyTavern, then go to **Extensions** > **Install Extension**
2. Paste: `https://github.com/chaaruze/SillyTavern-Too-Many-Chats`
3. Click **Install** and refresh

---

## Usage

### Create Folders

Click the **New Folder** button in the Chat History header.

### Organize Chats

Right-click any chat, then select a destination folder.

### Collapse or Expand

Click any folder header to toggle visibility.

### Manage Folders

Hover over a folder, then click the rename or delete icon.
